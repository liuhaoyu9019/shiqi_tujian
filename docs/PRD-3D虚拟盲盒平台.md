# 产品需求文档 (PRD) — 3D虚拟盲盒开盒平台

**版本**: v1.0
**日期**: 2026-05-15
**作者**: 产品经理（P8）

---

## 一、产品定位

纯虚拟盲盒开盒体验平台——用户付费购买虚拟盲盒，通过3D动画实时开盒揭晓藏品，收集稀有虚拟藏品，支持社交分享。无实体物流，纯数字体验。

## 二、目标用户

泛娱乐年轻用户（15-30岁），追求开盒惊喜感、虚拟收藏成就感、社交分享乐趣。

## 三、核心业务闭环

```
充值虚拟币 → 选购盲盒系列 → 支付解锁 → 3D开盒动画 → 揭晓藏品 → 收藏入册 → 社交分享 → 重复消费
```

## 四、功能全景

### 模块一：用户体系
- 手机号/微信登录注册
- 个人中心（头像、昵称、等级）
- 虚拟币钱包（充值/消费/余额）
- 充值套餐：¥10 / ¥30(+3) / ¥50(+8) / ¥100(+20) / ¥300(+90) / ¥500(+200)

### 模块二：盲盒商城
- 系列专区（主题分类、稀有度标识）
- 盲盒商品卡片（3D缩略图、价格、稀有款概率公示）
- 热门/新品标签筛选
- 单抽 / 5连抽 / 10连抽

### 模块三：3D开盒引擎（核心体验）
- 3D盲盒模型渲染（Three.js / WebGL）
- 开盒动画（旋转加速/光效/粒子特效）
- 藏品揭晓弹窗（稀有度闪光、震动反馈）
- 多种开盒交互（单击、连抽）

### 模块四：藏品系统
- 个人藏馆（3D展柜陈列）
- 藏品图鉴（系列收集进度、稀有度标识）
- 藏品3D预览（旋转/缩放/细节查看）
- 收藏成就系统（集齐奖励）

### 模块五：社交模块
- 开盒结果一键分享（生成海报/动图）
- 好友排行（收藏值/稀有藏品数）
- 社区晒盒（图文动态）

### 模块六：运营后台
- 盲盒系列管理（上架/下架/定价）
- 藏品CRUD（名称/图片/3D模型/稀有度/概率配置）
- 用户数据看板
- 订单/交易/充值管理
- 访问统计分析

## 五、平台覆盖

| 平台 | 3D能力 | 优先级 |
|------|--------|--------|

## 七、核心数据指标

- DAU、开盒转化率、人均开盒次数
- 藏品收集率、稀有款持有率
- 分享率、回流率、ARPU
- 充值转化率、客单价

## 八、设计原则

- **简洁**: 让3D盲盒成为绝对主角，界面退后、内容向前
- **明亮**: 白色为主基调，点缀活力色彩
- **清晰**: 信息层级分明，操作路径最短化
